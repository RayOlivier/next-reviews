# Indie Gamer Reviews

A video game review site built as a portfolio piece.
<br/>
 [Live Site](https://next-reviews-eta.vercel.app/)

## Description

This project is my first deep dive into Next.js and Tailwind CSS, using Strapi for headless CMS. 

### Dependencies

* React
* Next.js
* Tailwind CSS
* HeadlessUI (combobox)
